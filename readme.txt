== Description ==
A tool to dynamically display specific columns from any WordPress database table.

== Installation ==
1. Upload `database-table-display` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode `[display_table]` to display the table.

== Frequently Asked Questions ==

= How do I select the table to display? =
Go to the plugin settings page and select the table from the dropdown menu.
